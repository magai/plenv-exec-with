# NAME

plenv-exec-with - plenv exec with specified perl version

# SYNOPSIS

    plenv exec-with 5.14.4 cinnamon production deploy

# INSTALLATION

```sh
$ git clone https://github.com/magai/plenv-exec-with ~/.plenv/plugins/plenv-exec-with
```
